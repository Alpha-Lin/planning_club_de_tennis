# Planning club de tennis

Site développé pendant mon temps libre à destination du club de tennis de RS Saint Cyr sur Loire. Le site permet de gérer un planning de cours capable d'être modifiés à tout moment grâce à l'interface administrateur. La création et l'édition d'utilisateurs sont aussi permises via cette interface. De plus un système de "Vu" permet aux utilisateurs de signaler qu'ils ont bien été averti du nouveau cours ou de sa modification.
